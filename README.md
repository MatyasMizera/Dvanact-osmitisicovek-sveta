## **Název projektu**

**Dvanáct osmitisícovek světa**

## **Žák**

Matyáš Mizera

## **Stručný popis tématu (3–5 vět)**

Osmitisícovky jsou dvanáct nejvyšších hor světa, které se nacházejí v Himalájích a Karákóramu. Jejich výška přes 8000 metrů znamená extrémní podmínky, málo kyslíku a vysoké riziko. Přesto lákají horolezce svou náročností i působivou krásou.

## **Cílová skupina**

- **Horolezci a nadšenci do vysokohorské turistiky**
    
- **Lidé se zájmem o geografii a extrémní přírodní podmínky**

## **Návrh obsahových sekcí**


1. Základní informace o tom co je nad 8000m n. m.
    
2. O každé hoře Hlavní informace o nich, trasách a poloze
    
3. Nejvýznamnější horolezci, kteří je zdolali 
    

## **Návrh designu / wireframe**


- Barevné ladění v do bílo, černo, modré kombinace 
    
- Nějaké menu kde budou jednotlivé sekce(hory).
    
- Karty jednotlivých hor
    
- Jednoduchá úprava textu a záložek tak aby nebyl web zbytečně přeplacaný
    

_(Žák může doplnit ruční skicu, schéma nebo jednoduchý digitální wireframe.)_

## **Použité technologie**

- **Frontend:** HTML, CSS, JavaScript
    
- **Data:** JSON soubory pro čtyři textové sekce + JSON nabídka štěňat
    
- **AJAX:** načítání obsahu sekcí a seznamu štěňat
    
- **Backend:** pravděpodobně Python nebo PHP 
    
- **Verzování:** Git + GitHub (repozitář dostupný vyučujícímu)
